Slack Desk Bot

Internal desk & room booking bot for Slack

Python Slack Bolt SDK Socket Mode SQLite
Project
Overview

What the bot does

Desk Booking
Book an available desk in any office via slash command or the home tab
Room Booking
Reserve meeting rooms across 2–3 offices with time-slot validation
Occupancy View
See live occupancy bars per office on the Slack App Home tab
SQLite (WAL mode) — no external DB needed
Socket Mode — no public HTTP endpoint

Setup
Step 2 — Environment

Configure the .env file

Create a .env file with your Slack tokens:

# Slack Desk Bot — .env

SLACK_APP_TOKEN=xapp-1-xxxxxxxxxxxx
# App-Level Token → Features → Socket Mode

SLACK_BOT_TOKEN=xoxb-xxxxxxxxxxxx
# Bot Token → Features → OAuth & Permissions

SLACK_SIGNING_SECRET=xxxxxxxxxxxxxxx
# Signing Secret → Settings → Basic Information

Troubleshoot
Common Issues

Troubleshooting

not_enabled
App Home tab is not enabled. Go to Features → App Home and toggle "Always Show My Bot's Tabs" ON. Then reinstall the app.
invalid_auth / not_authed
Token is wrong or the app was uninstalled. Double-check the .env file and reinstall from OAuth & Permissions.
Home page blank
Check bot logs for not_enabled. If not, the tab filter might be blocking it (fixed in the current code).
Connection reset by peer
Socket Mode connection dropped. The SDK auto-reconnects, but check network stability and that the app-level token has connections:write scope.
Need to reinstall the app?
After changing scopes or settings, go to OAuth & Permissions → "Reinstall to Workspace". Tokens may change — update .env if so.
Virtual environment issues
If pip is missing: run python3 -m venv .venv --without-pip then python3 /tmp/get-pip.py.
